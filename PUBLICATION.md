## Package Description (Short - for Packagist)

**Laravel OpenProject Feedback** - A beautiful, customizable feedback widget for Laravel that automatically creates work packages in OpenProject. Collect user feedback, bug reports, and feature requests directly from your application.

## Package Description (Long - for README/Articles)

**Laravel OpenProject Feedback** is a powerful Laravel package that seamlessly integrates user feedback collection with OpenProject issue tracking. With a beautiful, customizable widget that appears on your pages, users can easily submit feedback, bug reports, and feature requests. The package automatically creates work packages in OpenProject, streamlining your feedback-to-issue workflow.

### Key Features:
- Zero-configuration setup with sensible defaults
- Fully customizable widget (position, colors, text)
- Screenshot support for visual feedback
- Automatic OpenProject work package creation
- Dark mode support
- Fully responsive design
- Optional authentication requirement
- Lightweight and performant
